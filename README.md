1) create a cluster 

```bash
./create_kind.sh
```

2) install flux v1 

```bash
./fluxv1.sh
```

3) install flu v2 

```bash
./fluxv2.sh 
```

3) scale down the flux operator and flux to 0 replicas 

4) reconcilie 

```bash
flux reconcile hr nginx -n default
flux reconcile kustomization flux-system --with-source
```

